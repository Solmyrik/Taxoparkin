Вы можете использовать метод "parallax" для этой цели. Вот пример кода с комментариями: ```html

Заголовок блока

``` В этом примере "parallax" достигается путем изменения свойства transform для внутреннего блока при скролле страницы. Значение transform translateY используется для перемещения блока вверх или вниз на определенное количество пикселей, определяемое значением scrollTop. Мы умножаем значение scrollTop на 0,5, чтобы сделать движение более плавным. Кроме того, для родительского блока установлено свойство "overflow: hidden", чтобы скрыть часть внутреннего блока, выходящую за пределы родительского блока при перемещении.
